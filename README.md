 # PreventivaPIM - Sistema de Manutenção Preventiva

Sistema web para **planejamento, execução e acompanhamento de manutenções preventivas industriais**.

O projeto foi desenvolvido como parte da trilha **Full Stack**, simulando um cenário real do **Polo Industrial de Manaus**, onde muitas empresas ainda controlam manutenções através de planilhas ou registros manuais.

A aplicação substitui esse processo por um **sistema digital com controle de equipamentos, planos de manutenção, calendário de execuções e dashboard de indicadores**.

---

# 🎯 Problema que o sistema resolve

A manutenção corretiva (quando a máquina já quebrou) gera:

- parada de produção
- custos emergenciais
- risco de acidentes
- perda de produtividade

A solução é a **manutenção preventiva**, que consiste em executar intervenções planejadas **antes da falha ocorrer**.

Este sistema permite:

- cadastrar equipamentos
- criar planos de manutenção com periodicidade
- registrar execuções de manutenção
- visualizar calendário de preventivas
- acompanhar indicadores de conformidade

---

# 🏭 Público alvo

O sistema foi projetado para três perfis principais:

### Técnico de manutenção
- visualiza manutenções programadas
- registra execução de manutenção
- adiciona observações técnicas

### Supervisor de manutenção
- cadastra planos de manutenção
- acompanha manutenções atrasadas
- monitora indicadores de conformidade

### Gestor de produção
- consulta histórico de manutenção
- acompanha disponibilidade de equipamentos

---
